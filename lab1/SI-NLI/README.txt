Description of attributes:

pair_id: unique identifier for instance
premise: premise sentence
hypothesis: hypothesis sentence
annotation_1: annotator_1 label
comment_1: annotator_1 comment
annotator_1_id: unique identifier for annotator_1
annotation_2: annotator_2 label
comment_2: annotator_2 comment
annotator_2_id: unique identifier for annotator_2
annotation_3: annotator_3 label
comment_3: annotator_3 comment
annotator_3_id: unique identifier for annotator_3
annotation_FINAL: label code
label: aggregated target class, derived from individual annotations; contains the relation between the premise and the hypothesis
